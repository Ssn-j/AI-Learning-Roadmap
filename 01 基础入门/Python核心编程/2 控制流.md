# 控制流

## 1 条件控制

Python 条件语句是通过一条或多条语句的执行结果（True 或者 False）来决定执行的代码块。

### 1.1 条件判断关键字

| 关键字 / 函数 | 分类   | 说明                     | 示例                     |
| -------- | ---- | ---------------------- | ---------------------- |
| `if`     | 条件判断 | 核心关键字                  | `if x > 0:`            |
| `elif`   | 条件判断 | 多分支（else if）           | `elif x == 0:`         |
| `else`   | 条件判断 | 所有条件不满足时执行             | `else:`                |
| `pass`   | 占位符  | 语法占位，保证语法完整            | `if x > 0: pass`       |
| `match`  | 模式匹配 | Python 3.10+，类似 switch | `match x: case 1: ...` |
| `case`   | 模式匹配 | match的分支               | `match x: case 1: ...` |

### 1.2 if语句

（1）if语句

语法格式：

```python
if 条件1:
    # 条件1为True时执行
elif 条件2:
    # 条件1为False且条件2为True时执行
else:
    # 以上条件都为False时执行
```

**注意：**

- 每个条件后面必须有冒号 `:`
- 条件可以是比较运算（`>`、`<`、`==`、`!=`、`>=`、`<=`）或逻辑运算（`and`、`or`、`not`）

（2）if嵌套

语法格式：

```python
if 表达式1:
    语句
    if 表达式2:
        语句
    elif 表达式3:
        语句
    else:
        语句
elif 表达式4:
    语句
else:
    语句
```

### 1.3 match...case

语法格式：

```python
match 要匹配的变量:
    case 匹配条件1:
        执行代码1
    case 匹配条件2:
        执行代码2
    case 匹配条件3:
        执行代码3
    case _:
        以上都不匹配时执行的代码ildcard>
```

`case _`: 类似于 C 和 Java 中的 default:，当其他 case 都无法匹配时，匹配这条，保证永远会匹配成功。

一个 case 也可以设置多个匹配条件，条件使用 `|` 隔开

```python
age = int(input("请输入年龄："))

match age:
    case 12:
        print('小学毕业啦！')
    case 15:
        print('初中毕业啦！')
    case 18:
        print('高中毕业啦！')
    case 22:
        print('大学毕业啦！')
    case _:
        print('年龄不在范围内！')
```

### 1.4 pass语句

（1）语法格式：

```python
if True:
    pass  # 占位，暂时不写代码
```

（2）常用场景：

- **代码占位**：先搭好结构，后续再补充逻辑，避免语法报错。

```python
for i in range(5):
    pass  # 循环体暂未编写
```

- **空函数 / 空类**：定义函数、类时，内部暂无代码。

```python
def test():
    pass

class Demo:
    pass
```

- **分支留空**：`if/elif/case` 等分支暂时不做处理

```python
match num:
    case 1:
        print("一")
    case 2:
        pass  # 此分支暂时无操作
    case _:
        print("其他")
```

## 2 循环语句

### 2.1 循环控制关键字与方法

| 关键字 / 函数    | 说明                   | 示例                               |
| ----------- | -------------------- | -------------------------------- |
| for         | 迭代循环，遍历序列或可迭代对象      | for i in list:                   |
| while       | 条件循环，条件为 True 时持续执行  | while x > 0:                     |
| break       | 立即终止当前循环             | break                            |
| continue    | 跳过本次循环剩余代码，进入下一轮迭代   | continue                         |
| else（循环）    | 循环正常结束（未触发 break）时执行 | for i in range(3): ... else: ... |
| pass        | 循环内占位空操作，保证语法完整      | for i in range(5): pass          |
| range()     | 生成整数序列，常搭配 for 循环使用  | range(0, 5)                      |
| enumerate() | 遍历同时获取索引与元素值         | for i, v in enumerate(list):     |

### 2.2 while循环

（1）语法格式：

```python
while 判断条件：
    执行语句……
```

（2）简单语句组

如果你的 while 循环体中只有一条语句，你可以将该语句与 while 写在同一行中

```python
while 判断条件：执行语句……
```

（3）while 循环使用 else 语句

如果 while 后面的条件语句为 false 时，则执行 else 的语句块。

- 语法格式：

```python
while 条件表达式:
    循环执行的语句
else:
    循环正常结束后执行的语句
```

- 示例：

```python
i = 1
while i <= 3:
    print(i)
    i += 1
else:
    print("循环正常执行完毕")
```

输出结果：

```python
1
2
3
循环正常执行完毕
```

（4）无限循环

实现：通过设置条件表达式永远不为 false 来实现无限循环

中断循环： **CTRL+C**

作用：无限循环在服务器上客户端的实时请求非常有用

```python
i = 1
while i == 1:print(i)
print("循环结束")
```

输出结果：

```
1
1
1
1
……
```

### 2.3 for语句

for 循环可以遍历任何可迭代对象，for...else用于在循环结束后执行一段代码

（1）语法格式：

```python
for 变量 in 序列:
    循环体语句
else:
    循环正常结束后执行的语句
```

作用：可用于打印字符串中的每个字符

```python
str = "hello"
for i in str:
    print(i)
```

输出结果：

```shell
h
e
l
l
o
```

（2）range()函数

- 语法格式：

```python
range(起始值, 结束值, 步长)
```

- **注意：**
  
  - 只提供一个参数：它将生成一个从 0 开始的整数序列，参数为结束值，步长默认为 1
  
  - 只提供开始值和结束值：那么步长默认为 1

- 作用：
  
  - 整数范围值可以配合 range() 函数使用
  
  ```python
  for i in range(2,9,2):
      print(i)
  ```
  
  输出结果：
  
  ```shell
  2
  4
  6
  8
  ```
  
  - 可以结合 range() 和 len() 函数以遍历一个序列的索引
  
  ```python
  tup = ('hi','my','name','is','sandeep')
  for i in range(len(tup)):
      print(i,tup[i])
  ```
  
  输出结果：
  
  ```shell
  0 hi
  1 my
  2 name
  3 is
  4 sandeep
  ```

（3）enumerate()

遍历时同时获取索引和值

```python
tup = ('hi','my','name','is','sandeep')
for i in enumerate(tup):
    print(i)
```

输出结果：

```shell
(0, 'hi')
(1, 'my')
(2, 'name')
(3, 'is')
(4, 'sandeep')
```

### 2.4 跳转语句

（1）**break** 语句（跳出语句）

终止当前的整个循环，剩余的循环代码（包括对应的循环 else 块）不再执行

```python
for i in range(1, 5):
    if i == 3:
        break
    print(i)
```

输出结果：

（2）**continue** 语句（继续语句）

跳过当前循环块中的剩余语句，然后继续进行下一轮循环。

```python
for i in range(1, 5):
    if i == 3:
        continue
    print(i)
```

输出结果：

```shell
1
2
4
```
