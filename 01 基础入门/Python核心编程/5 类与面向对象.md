# 类与面向对象

面向对象编程（OOP）是一种编程思想，把数据和对数据的操作封装在一起。类是蓝图，对象是根据蓝图创建出来的具体实例。

## 1. 类与对象

### 1.1 类的定义

**基本语法**：

```python
class 类名:
    定义方法、属性
```

### 1.2 `__init__` 构造函数

**作用**：在创建对象时**自动调用**，用于初始化对象的属性。

**示例**：

```python
class Student:
    def __init__(self, name, age):
        self.name = name   # 实例属性
        self.age = age

# 创建对象（实例化）
stu1 = Student("张三", 20)
stu2 = Student("李四", 22)

print(stu1.name)  # 输出：张三
print(stu2.age)   # 输出：22
```

### 1.3 `self` 的含义

**类的方法**与普通的函数只有一个特别的区别——它们必须有一个额外的**第一个参数名称** `self`，它指向实例本身，代表当前对象的地址

**注意**：`self`不是关键字，可以用别的名称表示

**示例：**

```python
class Test:
    def prt(self):
        print(self) # self 代表的是类的实例，代表当前对象的地址
        print(self.__class__) #  self.class 则指向类

t = Test()
t.prt()
```

执行结果：

```shell
# 实例地址
<__main__.Test object at 0x000002896D11A150>

# __main__ 当前正在运行的文件（主程序）本身
<class '__main__.Test'>
```

### 1.4 关键点

| 要素         | 说明               |
| ---------- | ---------------- |
| `class`    | 定义类的关键字          |
| `__init__` | 构造函数，创建对象时自动执行   |
| `self`     | 代表实例本身，是方法的第一个参数 |
| 实例化        | `对象 = 类名(参数)`    |

## 2. 实例方法

**定义**：在类的内部用 `def` 定义，方法**第一个参数必须为 `self`**，常规场景下仅通过实例调用。

```python
class Calculator:
    def add(self, a, b):
        return a + b

calc = Calculator()
print(calc.add(3, 5))       # 输出：8
```

**实例方法可以访问**：

- 实例属性（通过 `self.属性名`）
- 类属性（通过 `类名.属性名` 或 `self.__class__.属性名`）
- 其他实例方法（通过 `self.方法名()`）

```python
class Student:
    # 类属性
    school = "一中"

    # 构造方法（实例属性）
    def __init__(self, name):
        self.name = name  # 实例属性

    # 实例方法1
    def show(self):
        # 1. 访问 实例属性
        print(self.name)

        # 2. 访问 类属性
        print(Student.school)   # 类名访问
        print(self.__class__.school)  # self.__class__访问

        # 3. 调用 其他实例方法
        self.test()

    # 实例方法2
    def test(self):
        print("我是其他实例方法")

# 调用
s = Student("小明")
s.show()
```

运行结果：

```shell
小明
一中
一中
我是其他实例方法
```

## 3. 类属性

**定义**：直接定义在类内部的变量，所有实例共享同一个值。

**与实例属性的区别**：

| 类型   | 定义位置                  | 访问方式                  | 归属特点          |
| ---- | --------------------- | --------------------- | ------------- |
| 实例属性 | 方法内部使用 `self.属性名 = 值` | `实例对象.属性名`            | 每个实例独立拥有，互不共享 |
| 类属性  | 类内部、方法外部直接定义          | `类名.属性名` / `实例对象.属性名` | 所有实例共享同一个数据   |

**示例**：

```python
class Student:
    school = "清华大学"  # 类属性，所有实例共享

    def __init__(self, name):
        self.name = name  # 实例属性，每个实例独立

s1 = Student("张三")
s2 = Student("李四")

print(s1.school)  # 清华大学
print(s2.school)  # 清华大学

# 修改类属性会影响所有实例
Student.school = "北京大学"
print(s1.school)  # 北京大学
print(s2.school)  # 北京大学

# 实例属性是独立的
s1.name = "张三五"
print(s1.name)  # 张三五
print(s2.name)  # 李四
```

**典型使用场景**：计数器、配置常量、所有实例共享的数据。

```python
class Dog:
    count = 0  # 统计创建了多少只狗

    def __init__(self, name):
        self.name = name
        Dog.count += 1

d1 = Dog("旺财")
d2 = Dog("小黑")
print(Dog.count)  # 输出：2
```

## 4. 继承

子类继承父类的属性和方法，实现代码复用。

### 4.1 基本语法

```python
class 父类:
    pass

class 子类(父类):
    pass
```

**示例：**

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def eat(self):
        print(f"{self.name} 正在吃东西")

class Dog(Animal):
    def bark(self):
        print(f"{self.name} 汪汪叫")

class Cat(Animal):
    def meow(self):
        print(f"{self.name} 喵喵叫")

dog = Dog("旺财")
dog.eat()   # 继承自父类：旺财 正在吃东西
dog.bark()  # 自己的方法：旺财 汪汪叫

cat = Cat("咪咪")
cat.eat()   # 继承自父类：咪咪 正在吃东西
cat.meow()  # 自己的方法：咪咪 喵喵叫
```

### 4.2 方法重写

子类可以重写父类的方法。

```python
class Animal:
    def speak(self):
        print("动物在叫")

class Dog(Animal):
    def speak(self):  # 重写父类方法
        print("汪汪")

dog = Dog()
dog.speak()  # 输出：汪汪（不是"动物在叫"）
```

### 4.3 `super()` 调用父类

当子类需要扩展父类的方法时，用 `super()` 调用父类。

```python
class Animal:
    def __init__(self, name):
        self.name = name

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name)  # 调用父类的 __init__
        self.breed = breed      # 子类自己的属性

dog = Dog("旺财", "金毛")
print(dog.name)   # 输出：旺财（父类属性）
print(dog.breed)  # 输出：金毛（子类属性）
```

### 4.4 继承的关键点

| 概念        | 说明          |
| --------- | ----------- |
| 父类（基类）    | 被继承的类       |
| 子类（派生类）   | 继承出来的类      |
| 方法重写      | 子类重新实现父类的方法 |
| `super()` | 调用父类的方法     |

## 5. 封装

把属性和方法保护起来，不让外面随便改、随便访问。只留安全的接口给外面用。

### 5.1 封装的层次

| 命名方式     | 访问权限                    |
| -------- | ----------------------- |
| `name`   | 公开属性，外部可直接访问            |
| `_name`  | 约定为“受保护的”，外部可以访问但不建议    |
| `__name` | **私有属性**，外部无法直接访问（名称修饰） |

**示例：**

```python
class Student:
    def __init__(self, name, age, secret):
        self.name = name # 公开属性
        self._age = age # 受保护属性
        self.__secret = secret # 私有属性

student = Student("John", 18, "secret")
print(student.name)  # 输出: John
print(student._age)  # 输出: 18
print(student.__secret)  
# 输出: AttributeError: 'Student' object has no attribute '__secret'
```

### 5.2 属性装饰器

将方法伪装成属性形式访问，可在**读取 / 赋值**时增加校验、逻辑处理

#### 5.2.1常用组合说明

| 装饰器           | 名称          | 功能         | 使用场景          |
| ------------- | ----------- | ---------- | ------------- |
| `@property`   | getter（读取器） | 获取属性值      | `对象.属性名`（无括号） |
| `@属性名.setter` | setter（设置器） | 修改 / 赋值属性值 | `对象.属性名 = 新值` |

**注意：**

- `@属性名.setter` 必须和上方 `@property` 修饰的**方法名保持一致**。

- 只写 `@property`、不写 `setter`：该属性**只读**，无法外部赋值。

```python
class Person:
    def __init__(self, age):
        self.__age = age  # 私有属性

    # 读取值：getter
    @property
    def age(self):
        return self.__age

    # 赋值：setter，方法名必须和上方一致
    @age.setter
    def age(self, value):
        # 可自定义校验逻辑
        if value > 0:
            self.__age = value

# 调用演示
p = Person(18)
print(p.age)      # 读取，调用 getter
p.age = 25        # 赋值，调用 setter
print(p.age)
```

## 6. 特殊方法

以**双下划线开头和结尾**的方法，在特定操作时自动调用。

### 6.1 `__str__` 和 `__repr__`

| 方法         | 触发时机              | 用途                 |
| ---------- | ----------------- | ------------------ |
| `__str__`  | `print()`、`str()` | 给**用户**看的、友好的描述    |
| `__repr__` | 交互式环境、`repr()`    | 给**开发者**看的、可用来重建对象 |

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __str__(self):
        return f"学生：{self.name}，{self.age}岁"

    def __repr__(self):
        return f"Student('{self.name}', {self.age})"

stu = Student("张三", 20)
print(stu)           # 调用 __str__：学生：张三，20岁
print(repr(stu))     # 调用 __repr__：Student('张三', 20)
```

**最佳实践**：通常至少实现 `__str__`，方便打印调试。

### 6.2 其他常用特殊方法

| 方法            | 触发时机               | 用途                 |
| ------------- | ------------------ | ------------------ |
| `__len__`     | `len(obj)`         | 让对象支持 `len()` 计算长度 |
| `__getitem__` | `obj[key]`         | 让对象支持**索引 / 键取值**  |
| `__setitem__` | `obj[key] = value` | 让对象支持**索引 / 键赋值**  |
| `__call__`    | `obj()`            | 把**对象当函数直接调用**     |
| `__eq__`      | `obj1 == obj2`     | 自定义对象**相等判断规则**    |

```python
class Demo:
    def __init__(self):
        self.data = [1, 2, 3]  # 存放数据

    # 支持 len(obj)
    def __len__(self):
        return len(self.data)

    # 支持 obj[index] 取值
    def __getitem__(self, index):
        return self.data[index]

    # 支持 obj[index] = 值
    def __setitem__(self, index, value):
        self.data[index] = value

    # 支持 obj() 直接调用
    def __call__(self):
        return "对象被当作函数调用啦"

    # 支持 obj1 == obj2 判断
    def __eq__(self, other):
        return self.data == other.data

# 测试
d = Demo()
print(len(d))            # 输出：3  (__len__)
print(d[0])              # 输出：1  (__getitem__)
d[0] = 99                # 赋值操作
print(d())               # 输出：对象被当作函数调用啦  (__call__)

d2 = Demo()
print(d == d2)           # 输出：False  (__eq__)
```

## 7. 类方法 vs 静态方法

`self`（代表实例对象），`cls`（代表类本身）

| 类型   | 装饰器             | 第一个参数  | 调用方式   | 用途             |
| ---- | --------------- | ------ | ------ | -------------- |
| 实例方法 | 无，类内直接定义        | `self` | 通过实例   | 操作**实例属性**     |
| 类方法  | `@classmethod`  | `cls`  | 通过类或实例 | 操作**类属性**、工厂方法 |
| 静态方法 | `@staticmethod` | 无      | 通过类或实例 | 工具函数，逻辑上属于类    |

```python
class MyClass:
    class_var = 0

    def __init__(self, value):
        self.value = value

    # 实例方法：操作实例
    def instance_method(self):
        return self.value

    # 类方法：操作类属性
    @classmethod
    def class_method(cls):
        cls.class_var += 1
        return cls.class_var

    # 静态方法：工具函数
    @staticmethod
    def static_method(x, y):
        return x + y

obj = MyClass(10)

print(obj.instance_method())      # 10
print(MyClass.class_method())     # 1（通过类调用）
print(obj.class_method())         # 2（通过实例也可以）
print(MyClass.static_method(3, 5)) # 8
```

**使用建议**：

- 需要访问实例属性 → 实例方法
- 需要访问/修改类属性，不需要具体实例 → 类方法
- 只是逻辑上属于这个类的工具函数 → 静态方法

## 8. 综合示例：图书管理系统

```python
class Book:
    """图书类"""
    total_books = 0  # 类属性：总藏书量

    def __init__(self, title, author, isbn):
        self.title = title      # 公开属性
        self.author = author
        self.__isbn = isbn      # 私有属性
        self.is_borrowed = False
        Book.total_books += 1

    def borrow(self):
        """借书"""
        if not self.is_borrowed:
            self.is_borrowed = True
            print(f"成功借出《{self.title}》")
        else:
            print(f"《{self.title}》已被借出")

    def return_book(self):
        """还书"""
        if self.is_borrowed:
            self.is_borrowed = False
            print(f"成功归还《{self.title}》")
        else:
            print(f"《{self.title}》未被借出")

    @property
    def isbn(self):
        """通过property只读访问ISBN"""
        return self.__isbn

    def __str__(self):
        status = "已借出" if self.is_borrowed else "可借"
        return f"《{self.title}》- {self.author} [{status}]"


class EBook(Book):
    """电子书类，继承自Book"""
    def __init__(self, title, author, isbn, file_size):
        super().__init__(title, author, isbn)
        self.file_size = file_size

    def download(self):
        """电子书特有方法"""
        print(f"下载《{self.title}》，大小：{self.file_size}MB")


# 使用示例
book1 = Book("Python入门", "张三", "978-7-111-12345-6")
book2 = EBook("AI应用开发", "李四", "978-7-111-67890-1", 5.2)

book1.borrow()
book1.return_book()
print(book1)

book2.borrow()
book2.download()

print(f"总藏书量：{Book.total_books}")
```

## 9. 快速记忆表

| 概念              | 一句话          | 代码特征                   |
| --------------- | ------------ | ---------------------- |
| 类               | 蓝图/模板        | `class ClassName:`     |
| 对象              | 根据类创建的实例     | `obj = ClassName()`    |
| `__init__`      | 构造函数，初始化对象   | `def __init__(self):`  |
| `self`          | 指向实例本身       | 方法的第一个参数               |
| 实例属性            | 每个对象独立       | `self.xxx = value`     |
| 类属性             | 所有对象共享       | 直接写在类中                 |
| 继承              | 子类获得父类的属性和方法 | `class Child(Parent):` |
| `super()`       | 调用父类方法       | `super().__init__()`   |
| 私有属性            | 外部无法直接访问     | `__xxx`                |
| `@property`     | 把方法变成属性      | 装饰器                    |
| `__str__`       | `print()`时调用 | `def __str__(self):`   |
| `@classmethod`  | 类方法，操作类属性    | 第一个参数是 `cls`           |
| `@staticmethod` | 静态方法，工具函数    | 无特殊参数                  |
