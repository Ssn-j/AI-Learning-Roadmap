# 常用内置库

Python 自带丰富的标准库，不需要额外安装，直接 `import` 即可使用。

## 1  `os` 模块：操作系统接口

用于与操作系统交互（文件、目录、环境变量等）。

### 1.1 常用函数

| 函数                      | 说明           |
| ----------------------- | ------------ |
| `os.getcwd()`           | 获取当前工作目录     |
| `os.chdir(path)`        | 改变当前工作目录     |
| `os.listdir(path)`      | 列出目录下的文件和文件夹 |
| `os.mkdir(path)`        | 创建单级目录       |
| `os.makedirs(path)`     | 创建多级目录       |
| `os.remove(path)`       | 删除文件         |
| `os.rmdir(path)`        | 删除空目录        |
| `os.removedirs(path)`   | 删除多级空目录      |
| `os.rename(old, new)`   | 重命名文件或目录     |
| `os.path.exists(path)`  | 判断路径是否存在     |
| `os.path.isfile(path)`  | 判断是否为文件      |
| `os.path.isdir(path)`   | 判断是否为目录      |
| `os.path.join(a, b)`    | 拼接路径（跨平台）    |
| `os.path.getsize(path)` | 获取文件大小       |
| `os.environ`            | 环境变量字典       |

### 1.2 示例

```python
import os

# 1. 获取当前工作目录
print(os.getcwd())  # 输出：当前脚本所在的文件夹路径

# 2. 改变当前工作目录（切换到上一级目录）
os.chdir("..")
print(os.getcwd())  # 输出：切换后的路径

# 3. 列出当前目录下的文件和文件夹
print(os.listdir("."))  # 输出：[文件1, 文件夹1, test.py, ...]

# 4. 创建单级目录
os.mkdir("test_dir")  # 创建文件夹 test_dir

# 5. 创建多级目录（递归）
os.makedirs("a/b/c")  # 同时创建 a → b → c

# 6. 删除文件
# os.remove("test.txt")  # 删除文件 test.txt

# 7. 删除空目录
os.rmdir("test_dir")  # 删除空文件夹 test_dir

# 8. 删除多级空目录（递归删除）
os.removedirs("a/b/c")  # 删除 a/b/c 所有空目录

# 9. 重命名文件或目录
# os.rename("old.txt", "new.txt")  # 把 old 改成 new

# 10. 判断路径是否存在
print(os.path.exists("test.py"))  # 输出：True 或 False

# 11. 判断是否为文件
print(os.path.isfile("test.py"))  # 输出：True

# 12. 判断是否为目录
print(os.path.isdir("folder"))  # 输出：True 或 False

# 13. 拼接路径（自动适配系统）
path = os.path.join("folder", "sub", "file.txt")
print(path)  # 输出：folder/sub/file.txt（Linux/Mac）或 folder\sub\file.txt（Windows）

# 14. 获取文件大小（字节）
# print(os.path.getsize("test.py"))  # 输出：文件字节数

# 15. 环境变量字典
print(os.environ)  # 输出：系统所有环境变量
print(os.environ.get("PATH"))  # 输出：系统 PATH 环境变量
```

## 2. `sys` 模块：Python 解释器相关

**与 Python 解释器交互**：获取命令行参数、退出程序、查看环境、管理路径等。

### 2.1 常用属性

| 属性               | 说明                       |
| ---------------- | ------------------------ |
| `sys.argv`       | 命令行参数列表，`[0]` 是脚本名       |
| `sys.path`       | 模块搜索路径，可添加自定义路径          |
| `sys.platform`   | 系统平台（win32/linux/darwin） |
| `sys.version`    | Python 版本                |
| `sys.executable` | 解释器路径                    |
| `sys.stdin`      | 标准输入                     |
| `sys.stdout`     | 标准输出                     |
| `sys.stderr`     | 标准错误                     |

### 2.2 常用方法

| 方法                         | 说明            |
| -------------------------- | ------------- |
| `sys.exit(n)`              | 退出程序，0 = 正常退出 |
| `sys.getsizeof(obj)`       | 获取对象占用内存大小    |
| `sys.getdefaultencoding()` | 获取默认编码        |
| `sys.setrecursionlimit(n)` | 设置最大递归深度      |
| `sys.exc_info()`           | 获取当前异常信息      |

### 2.3 示例

```python
import sys

# ====================== 常用属性（按表格顺序）======================
# 1. sys.argv：命令行参数列表
print(sys.argv)  # 输出：['脚本名.py', 参数1, 参数2...]

# 2. sys.path：模块搜索路径
print(sys.path)  # 输出：Python 查找模块的路径列表

# 3. sys.platform：系统平台
print(sys.platform)  # 输出：win32 / linux / darwin

# 4. sys.version：Python 版本
print(sys.version)  # 输出：3.11.4 ...

# 5. sys.executable：解释器路径
print(sys.executable)  # 输出：python.exe 绝对路径

# 6. sys.stdin / sys.stdout / sys.stderr（标准流）
print(sys.stdout)  # 输出：标准输出对象

# 7. sys.byteorder：系统字节序
print(sys.byteorder)  # 输出：little

# 8. sys.maxsize：最大整数值
print(sys.maxsize)  # 输出：9223372036854775807


# ====================== 常用方法（按表格顺序）======================
# 1. sys.exit()：退出程序（0=正常）
# sys.exit(0)  # 执行后程序直接退出

# 2. sys.getsizeof()：获取对象占用内存（字节）
print(sys.getsizeof(100))  # 输出：28

# 3. sys.getdefaultencoding()：默认编码
print(sys.getdefaultencoding())  # 输出：utf-8

# 4. sys.setrecursionlimit()：设置递归深度
sys.setrecursionlimit(2000)  # 无输出，设置最大递归为2000

# 5. sys.getrecursionlimit()：获取递归深度
print(sys.getrecursionlimit())  # 输出：2000

# 6. sys.getrefcount()：对象引用次数
a = [1,2,3]
print(sys.getrefcount(a))  # 输出：2

# 7. sys.exc_info()：获取当前异常信息
print(sys.exc_info())  # 输出：(None, None, None)（无异常时）
```

## 3. `json` 模块：JSON 数据处理

JSON 是前后端数据交换的常用格式。

### 3.1 主要函数

| 函数                     | 作用                        |
| ---------------------- | ------------------------- |
| `json.dumps(obj)`      | **Python 转 JSON 字符串**（编码） |
| `json.loads(str)`      | **JSON 字符串 转 Python**（解码） |
| `json.dump(obj, file)` | Python 数据 **写入 JSON 文件**  |
| `json.load(file)`      | 从 **JSON 文件 读取** 转 Python |

### 3.2 类型映射

                **Python → JSON**                   **JSON → Python**

| Python 类型（转） | JSON 类型 | JSON 类型（转） | Python 类型 |
| ------------ | ------- | ---------- | --------- |
| dict         | object  | object     | dict      |
| list、tuple   | array   | array      | list      |
| str          | string  | string     | str       |
| int、float    | number  | number     | int、float |
| True         | true    | true       | True      |
| False        | false   | false      | False     |
| None         | null    | null       | None      |

### 3.3 示例

```python
import json

# 1. Python 转 JSON 字符串
data = {"name": "小明", "age": 20}
json_str = json.dumps(data)  # 编码

# 2. JSON 字符串 转 Python
py_data = json.loads(json_str)  # 解码

# 3. 写入 JSON 文件
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False)

# 4. 读取 JSON 文件
with open("data.json", "r", encoding="utf-8") as f:
```

## 4. `datetime` 模块：日期时间处理

### 4..1 主要类

| 类           | 说明           |
| ----------- | ------------ |
| `datetime`  | 日期+时间        |
| `date`      | 日期（年-月-日）    |
| `time`      | 时间（时:分:秒.微秒） |
| `timedelta` | 时间差（用于加减）    |

### 4.2 常用格式化符号

| 格式码  | 说明        | 示例     |
| ---- | --------- | ------ |
| `%Y` | 4位年份      | 2026   |
| `%m` | 2位月份      | 05     |
| `%d` | 2位日期      | 30     |
| `%H` | 24小时制小时   | 14     |
| `%M` | 分钟        | 30     |
| `%S` | 秒         | 00     |
| `%f` | 微秒        | 123456 |
| `%w` | 星期几（0=周日） | 6      |
| `%j` | 年内的第几天    | 150    |

### 4.3 常用操作

| 操作     | 代码                               | 说明          |
| ------ | -------------------------------- | ----------- |
| 获取当前时间 | `datetime.now()`                 | 获取当前日期时间    |
| 创建指定时间 | `datetime(2025,4,22,15,30)`      | 手动创建日期时间    |
| 格式化输出  | `.strftime("%Y-%m-%d %H:%M:%S")` | 时间 → 字符串    |
| 字符串转时间 | `datetime.strptime(s, fmt)`      | 字符串 → 时间    |
| 时间加减   | `now + timedelta(days=7)`        | 计算未来 / 过去时间 |
| 计算时间差  | `(d2 - d1).days`                 | 获取相差天数      |

### 4.4 示例：

```python
from datetime import datetime, timedelta

# 1. 获取当前时间
now = datetime.now()
print(now)  # 输出当前日期时间

# 2. 创建指定时间
dt = datetime(2025, 4, 22, 15, 30, 0)
print(dt)  # 输出 2025-04-22 15:30:00

# 3. 时间格式化为字符串
str_time = now.strftime("%Y-%m-%d %H:%M:%S")
print(str_time)  # 输出 2025-xx-xx xx:xx:xx

# 4. 字符串转时间对象
dt2 = datetime.strptime("2025-04-22", "%Y-%m-%d")
print(dt2)  # 输出 2025-04-22 00:00:00

# 5. 时间加减（7天后）
future = now + timedelta(days=7)
print(future)  # 输出一周后的日期时间

# 6. 计算两个日期相差天数
d1 = datetime(2025, 4, 22)
d2 = datetime(2025, 5, 1)
diff_days = (d2 - d1).days
print(diff_days)  # 输出 9
```

## 6. `random` 模块：随机数

### 6.1 常用函数表格

| 函数                           | 说明                     |
| ---------------------------- | ---------------------- |
| `random.random()`            | 生成 0.0 ~ 1.0 之间的随机浮点数  |
| `random.uniform(1, 10)`      | 生成 1.0 ~ 10.0 之间的随机浮点数 |
| `random.randint(1, 10)`      | 生成 1 ~ 10 之间的整数（包含两端）  |
| `random.randrange(0, 10, 2)` | 从 0,2,4,6,8 中随机取一个整数   |
| `random.choice(seq)`         | 从序列中随机选 1 个元素          |
| `random.sample(seq, 2)`      | 从序列中随机选 2 个不重复的元素      |
| `random.choices(seq, k=2)`   | 从序列中随机选 2 个可重复的元素      |
| `random.shuffle(lst)`        | 原地打乱列表顺序               |
| `random.seed(42)`            | 设置随机种子，让结果可复现          |

### 6.2 示例

```python
import random

# 生成随机数

random.random() # 0.0 ~ 1.0 之间的浮点数
random.uniform(1, 10) # 1.0 ~ 10.0 之间的浮点数
random.randint(1, 10) # 1 ~ 10 之间的整数（包含两端）
random.randrange(0, 10, 2) # 0,2,4,6,8 中的随机数

# 从序列中随机选择

random.choice(["a", "b", "c"]) # 随机选一个
random.sample(["a", "b", "c", "d"], 2) # 随机选2个（不重复）
random.choices(["a", "b", "c"], k=2) # 随机选2个（可重复）

# 打乱顺序

lst = [1, 2, 3, 4, 5]
random.shuffle(lst) # lst 顺序被打乱

# 设置随机种子（使结果可复现）

random.seed(42)
```

## 7. `re` 模块：正则表达式（了解即可）

**用于字符串匹配、查找、替换、分割**，是处理文本最强大的标准库。

### 7.1 常用函数表格

| 函数             | 作用              |
| -------------- | --------------- |
| `re.match()`   | 从**开头**匹配       |
| `re.search()`  | 搜索**第一个**匹配     |
| `re.findall()` | 查找**所有**匹配，返回列表 |
| `re.sub()`     | 替换匹配内容          |
| `re.split()`   | 按规则分割字符串        |
| `re.compile()` | 预编译正则，提升效率      |

### 7.2 常用元字符

| 元字符  | 作用              |
| ---- | --------------- |
| `\d` | 匹配数字            |
| `\w` | 匹配字母 / 数字 / 下划线 |
| `\s` | 匹配空白            |
| `.`  | 匹配任意字符          |
| `+`  | 匹配 1 次或多次       |
| `*`  | 匹配 0 次或多次       |
| `^`  | 匹配开头            |
| `$`  | 匹配结尾            |
| `[]` | 字符集             |
| `()` | 分组提取            |

### 7.3 示例

函数：

```python
import re

# 1. re.match：从开头匹配
print(re.match(r"hello", "hello world"))  # 匹配成功，返回对象

# 2. re.search：搜索第一个匹配
print(re.search(r"world", "hello world").group())  # 输出：world

# 3. re.findall：提取所有数字
print(re.findall(r"\d+", "age18, score99"))  # 输出：['18','99']

# 4. re.sub：替换字符串
print(re.sub(r"apple", "pear", "I like apple"))  # 输出：I like pear

# 5. re.split：按数字分割
print(re.split(r"\d+", "python123java456"))  # 输出：['python','java','']

# 6. re.compile：预编译
pat = re.compile(r"\w+")
print(pat.findall("hello 123"))  # 输出：['hello','123']
```

元字符：

```python
import re

text = "abc123 ABC_  hello"

# 1. \d  匹配数字
print(re.findall(r"\d", text))        # ['1', '2', '3']

# 2. \w  匹配字母/数字/下划线
print(re.findall(r"\w+", text))      # ['abc123', 'ABC_', 'hello']

# 3. \s  匹配空白
print(re.findall(r"\s", text))       # [' ', ' ', ' ']

# 4. .   匹配任意一个字符
print(re.findall(r"a.c", "abc adc")) # ['abc', 'adc']

# 5. +   1次或多次
print(re.findall(r"\d+", "a11b222")) # ['11', '222']

# 6. *   0次或多次
print(re.findall(r"ab*", "a ab abb")) # ['a', 'ab', 'abb']

# 7. ?   0次或1次
print(re.findall(r"ab?", "a ab abb"))# ['a', 'ab', 'ab']

# 8. ^   开头匹配
print(re.match(r"^abc", text))       # 匹配成功（对象）

# 9. $   结尾匹配
print(re.search(r"hello$", text))    # 匹配成功（对象）

# 10. [] 字符集
print(re.findall(r"[a-c]", text))    # ['a', 'b', 'c']

# 11. () 分组提取
print(re.search(r"(\d+)", text).group(1)) # '123'
```

## 8. `hashlib` 模块：哈希加密

### 8.1 模块作用

实现**哈希加密**，常用于数据完整性校验、密码加密存储；输入任意长度数据，输出固定长度哈希值。

**注意：** 传入数据必须为**字节类型 (bytes)**，字符串需先用 `encode()` 转换。

### 8.2 常用哈希算法对比表

| 算法     | 摘要长度 (位) | 安全性 | 适用场景          |
| ------ | -------- | --- | ------------- |
| md5    | 128      | 不安全 | 仅数据校验，不建议存密码  |
| sha1   | 160      | 不安全 | 老旧项目数据校验      |
| sha256 | 256      | 中等  | 通用加密、数据校验（推荐） |
| sha512 | 512      | 高   | 高安全需求、数字签名    |

### 8.3 核心方法

| 方法                  | 作用                  |
| ------------------- | ------------------- |
| `hashlib.算法名(字节数据)` | 直接创建哈希对象            |
| `hashlib.new(算法名)`  | 通用方式创建哈希对象          |
| `哈希对象.update(字节数据)` | 分段更新加密内容            |
| `哈希对象.hexdigest()`  | 返回**十六进制**哈希字符串（常用） |
| `哈希对象.digest()`     | 返回**原始二进制**哈希结果     |

### 8.4 示例

```python
import hashlib

# 待加密字符串（必须转成 bytes）
data = b"RUNOOB"

# ===================== 1. 按算法表格顺序加密 =====================
# MD5
md5 = hashlib.md5(data)
print(md5.hexdigest())  # 输出：18fa661e2a4a7dd6471cc1407290cf6e

# SHA1
sha1 = hashlib.sha1(data)
print(sha1.hexdigest())  # 输出：341709d0dbc266332f1cbcd8c25fd4a05412b7d4

# SHA256
sha256 = hashlib.sha256(data)
print(sha256.hexdigest())  # 输出：673dc967d03201db7fe47b7eabd56c47ca5bc694222de303106a5504e5d0daa8

# SHA512
sha512 = hashlib.sha512(data)
print(sha512.hexdigest())  # 输出：7cfe50493eebd48ee7330c797459c2d0d5ca943bd1c84ad7a0b6783b11cd49d06b4a1dc84ee9ea5e20d0bfedbdb67e716500a20e5870abecea3f32dc8484a811


# ===================== 2. 按核心方法表格顺序演示 =====================
# 1. 创建哈希对象
h = hashlib.sha256()

# 2. update() 添加数据
h.update(b"RUNOOB")

# 3. hexdigest() 获取十六进制结果
print(h.hexdigest())  # 输出：673dc967d03201db7fe47b7eabd56c47ca5bc694222de303106a5504e5d0daa8

# 4. digest() 获取二进制结果
print(h.digest())  # 输出：bytes 类型二进制哈希值

# 5. new() 通用创建方式
h2 = hashlib.new("sha256", b"RUNOOB")
print(h2.hexdigest())  # 输出同上 sha256 结果
```

示例：

```python
import hashlib

# MD5（不推荐用于密码）

md5 = hashlib.md5("password".encode()).hexdigest()

# SHA256

sha256 = hashlib.sha256("password".encode()).hexdigest()
```
