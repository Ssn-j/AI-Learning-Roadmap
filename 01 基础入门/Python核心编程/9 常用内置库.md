# 常用内置库

Python 自带丰富的标准库，不需要额外安装，直接 `import` 即可使用。

## 1. `os` 模块：操作系统接口

用于与操作系统交互（文件、目录、环境变量等）。

### 常用函数

| 函数                      | 说明           |
| ----------------------- | ------------ |
| `os.getcwd()`           | 获取当前工作目录     |
| `os.chdir(path)`        | 改变当前工作目录     |
| `os.listdir(path)`      | 列出目录下的文件和文件夹 |
| `os.mkdir(path)`        | 创建单级目录       |
| `os.makedirs(path)`     | 创建多级目录       |
| `os.remove(path)`       | 删除文件         |
| `os.rmdir(path)`        | 删除空目录        |
| `os.removedirs(path)`   | 删除多级空目录      |
| `os.rename(old, new)`   | 重命名文件或目录     |
| `os.path.exists(path)`  | 判断路径是否存在     |
| `os.path.isfile(path)`  | 判断是否为文件      |
| `os.path.isdir(path)`   | 判断是否为目录      |
| `os.path.join(a, b)`    | 拼接路径（跨平台）    |
| `os.path.getsize(path)` | 获取文件大小       |
| `os.environ`            | 环境变量字典       |


示例：

```python
import os

# 当前目录

current = os.getcwd()
print(current)

# 遍历目录

for item in os.listdir("."):
 print(item)

# 创建目录

os.makedirs("data/backup", exist_ok=True)

# 拼接路径

file_path = os.path.join("data", "backup", "file.txt")
print(file_path) # data/backup/file.txt（Windows 是 data\backup\file.txt）

# 环境变量

print(os.environ.get("PATH"))
```



## 2. `sys` 模块：Python 解释器相关

用于访问 Python 解释器的变量和函数。

### 常用属性/函数

| 属性/函数          | 说明          |
| -------------- | ----------- |
| `sys.argv`     | 命令行参数列表     |
| `sys.path`     | 模块搜索路径列表    |
| `sys.exit()`   | 退出程序        |
| `sys.version`  | Python 版本信息 |
| `sys.platform` | 操作系统平台      |
| `sys.stdin`    | 标准输入        |
| `sys.stdout`   | 标准输出        |
| `sys.stderr`   | 标准错误        |

示例：

```python
import sys

# 命令行参数

# 运行：python script.py arg1 arg2

print(sys.argv) # ['script.py', 'arg1', 'arg2']

# 模块搜索路径

print(sys.path)

# 平台信息

print(sys.platform) # 'win32' / 'linux' / 'darwin'

# 退出程序

if error:
 sys.exit(1) # 非0表示错误退出
```



## 3. `json` 模块：JSON 数据处理

JSON 是前后端数据交换的常用格式。

### 主要函数

| 函数                     | 说明                   |
| ---------------------- | -------------------- |
| `json.dumps(obj)`      | Python 对象 → JSON 字符串 |
| `json.loads(str)`      | JSON 字符串 → Python 对象 |
| `json.dump(obj, file)` | Python 对象 → JSON 文件  |
| `json.load(file)`      | JSON 文件 → Python 对象  |

### 类型映射

| Python      | JSON   |
| ----------- | ------ |
| dict        | object |
| list, tuple | array  |
| str         | string |
| int, float  | number |
| True        | true   |
| False       | false  |
| None        | null   |

示例：

```python
import json

# 字典转 JSON 字符串

data = {"name": "张三", "age": 20, "city": "北京"}
json_str = json.dumps(data, ensure_ascii=False, indent=2)
print(json_str)

# JSON 字符串转字典

json_str = '{"name": "李四", "age": 25}'
data = json.loads(json_str)
print(data["name"]) # 李四

# 写入 JSON 文件

with open("data.json", "w", encoding="utf-8") as f:
 json.dump(data, f, ensure_ascii=False, indent=2)

# 读取 JSON 文件

with open("data.json", "r", encoding="utf-8") as f:
 data = json.load(f)
```



## 4. `datetime` 模块：日期时间处理

### 4..1 主要类

| 类           | 说明           |
| ----------- | ------------ |
| `datetime`  | 日期+时间        |
| `date`      | 日期（年-月-日）    |
| `time`      | 时间（时:分:秒.微秒） |
| `timedelta` | 时间差（用于加减）    |

### 4.2 常用格式化符号

| 格式码  | 说明        | 示例     |
| ---- | --------- | ------ |
| `%Y` | 4位年份      | 2026   |
| `%m` | 2位月份      | 05     |
| `%d` | 2位日期      | 30     |
| `%H` | 24小时制小时   | 14     |
| `%M` | 分钟        | 30     |
| `%S` | 秒         | 00     |
| `%f` | 微秒        | 123456 |
| `%w` | 星期几（0=周日） | 6      |
| `%j` | 年内的第几天    | 150    |

### 4.3 常用操作

| 操作     | 代码                               | 说明          |
| ------ | -------------------------------- | ----------- |
| 获取当前时间 | `datetime.now()`                 | 获取当前日期时间    |
| 创建指定时间 | `datetime(2025,4,22,15,30)`      | 手动创建日期时间    |
| 格式化输出  | `.strftime("%Y-%m-%d %H:%M:%S")` | 时间 → 字符串    |
| 字符串转时间 | `datetime.strptime(s, fmt)`      | 字符串 → 时间    |
| 时间加减   | `now + timedelta(days=7)`        | 计算未来 / 过去时间 |
| 计算时间差  | `(d2 - d1).days`                 | 获取相差天数      |

示例：

```python
from datetime import datetime, timedelta

# 1. 获取当前时间
now = datetime.now()
print(now)  # 输出当前日期时间

# 2. 创建指定时间
dt = datetime(2025, 4, 22, 15, 30, 0)
print(dt)  # 输出 2025-04-22 15:30:00

# 3. 时间格式化为字符串
str_time = now.strftime("%Y-%m-%d %H:%M:%S")
print(str_time)  # 输出 2025-xx-xx xx:xx:xx

# 4. 字符串转时间对象
dt2 = datetime.strptime("2025-04-22", "%Y-%m-%d")
print(dt2)  # 输出 2025-04-22 00:00:00

# 5. 时间加减（7天后）
future = now + timedelta(days=7)
print(future)  # 输出一周后的日期时间

# 6. 计算两个日期相差天数
d1 = datetime(2025, 4, 22)
d2 = datetime(2025, 5, 1)
diff_days = (d2 - d1).days
print(diff_days)  # 输出 9
```

## 6. `random` 模块：随机数

### 6.1 常用函数表格

| 函数                           | 说明                     |
| ---------------------------- | ---------------------- |
| `random.random()`            | 生成 0.0 ~ 1.0 之间的随机浮点数  |
| `random.uniform(1, 10)`      | 生成 1.0 ~ 10.0 之间的随机浮点数 |
| `random.randint(1, 10)`      | 生成 1 ~ 10 之间的整数（包含两端）  |
| `random.randrange(0, 10, 2)` | 从 0,2,4,6,8 中随机取一个整数   |
| `random.choice(seq)`         | 从序列中随机选 1 个元素          |
| `random.sample(seq, 2)`      | 从序列中随机选 2 个不重复的元素      |
| `random.choices(seq, k=2)`   | 从序列中随机选 2 个可重复的元素      |
| `random.shuffle(lst)`        | 原地打乱列表顺序               |
| `random.seed(42)`            | 设置随机种子，让结果可复现          |

### 6.2 示例

```python
import random

# 生成随机数

random.random() # 0.0 ~ 1.0 之间的浮点数
random.uniform(1, 10) # 1.0 ~ 10.0 之间的浮点数
random.randint(1, 10) # 1 ~ 10 之间的整数（包含两端）
random.randrange(0, 10, 2) # 0,2,4,6,8 中的随机数

# 从序列中随机选择

random.choice(["a", "b", "c"]) # 随机选一个
random.sample(["a", "b", "c", "d"], 2) # 随机选2个（不重复）
random.choices(["a", "b", "c"], k=2) # 随机选2个（可重复）

# 打乱顺序

lst = [1, 2, 3, 4, 5]
random.shuffle(lst) # lst 顺序被打乱

# 设置随机种子（使结果可复现）

random.seed(42)
```



## 7. `re` 模块：正则表达式（了解即可）

示例：

```python
import re

# 查找匹配

pattern = r"\d+" # 匹配一个或多个数字
text = "我有10个苹果，20个香蕉"
result = re.findall(pattern, text)
print(result) # ['10', '20']

# 替换

new_text = re.sub(r"\d+", "X", text)
print(new_text) # 我有X个苹果，X个香蕉
```
