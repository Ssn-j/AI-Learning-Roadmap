# 函数

函数是组织代码的基本单元，把一段重复使用的逻辑封装起来，需要时调用即可。

## 1. 定义函数

### 1.1 基本语法

```python
def 函数名（参数列表）:
    函数体
```

### 1.2 定义规则

| 要素       | 说明                                  |
| -------- | ----------------------------------- |
| `def`    | 定义函数的关键字                            |
| 函数名      | 遵循变量命名规则，常用小写加下划线                   |
| 参数       | 可无参数，也可设置多个参数                       |
| `return` | 用于返回值，执行到此函数结束；无`return`时默认返回`None` |
| 文档字符串    | 存放函数说明                              |

**示例**：

```python
def add(a, b):
    """计算两个数的和"""
    return a + b

result = add(3, 5) # 函数调用
print(result)  # 输出：8
```

## 2.参数传递

Python 只有**对象引用传递**，是否改变外部变量，只看对象**可变 / 不可变**。

（1）规则：

| 传递对象  | 机制    | 效果              |
| ----- | ----- | --------------- |
| 不可变类型 | 传值效果  | 函数内修改不影响外部原变量   |
| 可变类型  | 传引用效果 | 函数内修改会同步影响外部原变量 |

（2）核心概念

- **对象有类型，变量无类型**：变量只是对象的引用（指针）
- **可变对象**：list、dict、set（可直接修改内容）
- **不可变对象**：int、float、str、tuple（无法修改，只能新建）

（3）示例

不可变对象（int/str）：

```python
def func(a):
    a = 10  # 新建对象，不影响外部
x = 1
func(x)
print(x)  # 1（没变）
```

可变对象（list/dict）：

```python
def func(lst):
    lst.append(4)  # 修改原对象
mylist = [1,2,3]
func(mylist)
print(mylist)  # [1,2,3,4]（变了）
```

## 3 参数类型

### 3.1 位置参数

调用时按位置传递，顺序必须一致。

```python
def greet(name, message):
    print(f"{message}, {name}")

greet("张三", "你好")  # 正确：输出"你好, 张三"
greet("你好", "张三")  # 错误：位置错乱
```

### 3.2 默认参数

在定义时给参数指定默认值，调用时可不传。

```python
def greet(name, message="你好"):
    print(f"{message}, {name}")

greet("张三")           # 使用默认：你好, 张三
greet("张三", "欢迎")   # 覆盖默认：欢迎, 张三
```

**注意**：默认参数必须放在位置参数后面。

### 3.3 关键字参数

调用时指定参数名，顺序可任意。

```python
def introduce(name, age, city):
    print(f"{name}, {age}岁, 来自{city}")

introduce(city="北京", name="张三", age=20)
```

### 3.4 不定长参数

#### 3.4.1 `*args`

接收**多个位置参数**，打包成**元组**。

```python
def func(*args):
    print(args)

func(1,2,3,"a")  # 输出：(1, 2, 3, 'a')
```

#### 3.4.2 `**kwargs`

接收任意**多个关键字参数**，打包成**字典**。

```python
def func(**kwargs):
    print(kwargs)

func(name="张三", age=18)  # 输出：{'name': '张三', 'age': 18}
```

### 3.5 参数优先级（从前往后）

```
位置参数 → 默认参数 → *args → **kwargs
```

```python
def func(a, b=1, *args, **kwargs):
    pass
```

## 4. 返回值 return

### 4.1 返回单个值

```python
def square(x):
    return x * x

result = square(5)  # result = 25
```

### 4.2 返回多个值

实际返回一个元组。

```python
def get_user():
    name = "张三"
    age = 20
    return name, age

name, age = get_user()  # 解包赋值
print(name, age)        # 输出：张三 20

# 也可以用一个变量接收
result = get_user()
print(result)  # 输出：('张三', 20)
```

### 4.3 无返回值

没有`return`或只写`return`，函数返回`None`。

```python
def say_hello(name):
    print(f"你好, {name}")
    # 没有return

result = say_hello("张三")
print(result)  # 输出：None
```

## 5. 文档字符串（docstring）

**作用**：为函数写说明文档，可以用 `help()` 查看。

**写法**：用三个双引号 `"""`。

**多行文档字符串格式**：

```python
def complex_function(param1, param2):
    """
    一句话概括函数功能。

    详细描述（可选）。

    参数:
        param1 (int): 参数1的说明
        param2 (str): 参数2的说明

    返回:
        bool: 返回值的说明

    异常:
        ValueError: 什么情况会抛出异常
    """
    pass
```

示例：

```python
def add(a, b):
    """
    计算两个数的和。

    参数:
        a: 第一个数
        b: 第二个数

    返回:
        两个数的和
    """
    return a + b

# 查看文档
help(add)
```

## 6 作用域

### 6.1 局部变量

```python
def my_func():
    x = 10  # 局部变量
    print(x)

my_func()
print(x)  # 报错：NameError，x不存在
```

### 6.2 全局变量

```python
name = "张三"  # 全局变量

def greet():
    print(name)  # 可以读取

greet()  # 输出：张三
```

### 6.3 修改全局变量：`global`

```python
count = 0

def increment():
    global count  # 声明要修改全局变量
    count += 1

increment()
print(count)  # 输出：1
```

### 6.4 嵌套作用域：`nonlocal`

用于在内层函数中修改外层函数的变量。

```python
def outer():
    x = 10
    def inner():
        nonlocal x  # 修改外层函数的x
        x += 5
    inner()
    print(x)  # 输出：15

outer()
```

## 6 作用域与命名空间

作用域是变量的可见范围，命名空间是存储变量名的容器。两者紧密相关：每个作用域都有对应的命名空间。

### 6.1 作用域（Scope）

**定义**：变量在程序中可以被访问的范围。

#### 6.1.1 种类（LEGB 规则）

| 作用域       | 英文        | 说明                            | 生命周期                 |
| --------- | --------- | ----------------------------- | -------------------- |
| **局部作用域** | Local     | 函数内部的变量                       | 函数调用时创建，返回时销毁        |
| **嵌套作用域** | Enclosing | 外层函数（非全局）的变量                  | 外层函数调用时创建，内层函数被销毁时销毁 |
| **全局作用域** | Global    | 模块/文件级别的变量                    | 模块被导入时创建，程序结束时销毁     |
| **内置作用域** | Built-in  | Python 自带的名称（如 `print`、`len`） | 解释器启动时创建，关闭时销毁       |

#### 6.1.2 变量查找顺序：LEGB 规则

Python 查找变量时，按 **L → E → G → B** 的顺序依次查找，找到就停止。

```text
Local（局部）→ Enclosing（闭包）→ Global（全局）→ Built-in（内置）
```

示例：

```python
x = "global" # 全局作用域
def outer():
 x = "enclosing" # 嵌套作用域
    def inner():
        x = "local"   # 局部作用域
        print(x)      # 找局部：找到 "local"
        # 如果注释掉局部 x，会找嵌套：找到 "enclosing"
        # 如果嵌套也没有，会找全局：找到 "global"
        # 如果全局也没有，会找内置
        # 如果都没有，报 NameError

    inner()
outer()
```

#### 6.1.3 变量

| 类型   | 定义位置 | 访问范围     |
| ---- | ---- | -------- |
| 局部变量 | 函数内部 | 只能在函数内访问 |
| 全局变量 | 函数外部 | 整个文件可访问  |

#### 6.1.4 变量修改规则

| 操作             | 规则            | 代码           |
| -------------- | ------------- | ------------ |
| 读取变量           | 按 LEGB 顺序查找   | `print(x)`   |
| 赋值变量           | 默认在当前作用域创建/修改 | `x = 10`     |
| 修改**嵌套作用域**的变量 | 使用 `nonlocal` | `nonlocal x` |
| 修改**全局变量**     | 使用 `global`   | `global x`   |

**示例：**

```python
x = 100  # 全局变量

def outer():
    x = 50  # 嵌套变量

    def inner():
        nonlocal x   # 声明要修改嵌套的 x
        global y     # 声明要修改全局的 y
        x = 200      # 修改嵌套的 x
        y = 300      # 修改全局的 y

    inner()
    print(x)  # 输出：200

outer()
print(y)  # 输出：300
print(x)  # 输出：100（全局变量没有变）
```

### 6.2 命名空间（Namespace）

**定义：** 命名空间是**变量名与对象的映射关系**，可以理解为“存储变量名的地方”。Python 中的每个作用域都有自己独立的命名空间。

#### 6.2.1 命名空间的类型

| 命名空间     | 对应作用域   | 说明                     |
| -------- | ------- | ---------------------- |
| 内置命名空间   | 内置作用域   | 存储 `print`、`len` 等内置名称 |
| 全局命名空间   | 全局作用域   | 存储模块级别的变量和函数           |
| 局部命名空间   | 局部作用域   | 存储函数内部的变量              |
| 闭包命名空间   | 嵌套作用域   | 存储外层函数中被内层函数引用的变量      |
| 类/实例命名空间 | 类/实例作用域 | 存储类的属性和实例的属性           |

#### 6.2.2 查看命名空间

```python
# 查看全局命名空间
print(globals())

# 查看局部命名空间
def func():
    a = 1
    b = 2
    print(locals())

func()  # 输出：{'a': 1, 'b': 2}

# 查看内置命名空间
print(dir(__builtins__))  # 列出所有内置名称
```

### 6.3 作用域 vs 命名空间

| 概念       | 本质                   | 关系          |
| -------- | -------------------- | ----------- |
| **作用域**  | 变量的**可见范围**（抽象规则）    | 定义“在哪里可以访问” |
| **命名空间** | 存储变量名的**具体容器**（实现机制） | 实现“变量存在哪里”  |

## 7. lambda 表达式

### 7.1 定义

**匿名函数**：无需 `def` 命名，单行快速定义

**特点**：多参数、单表达式、自动返回结果

**语法**：

```python
lambda 参数: 返回值
```

### 7.2 基础用法：

```python
# 无参
f = lambda: "Hello"
print(f())  # 输出: Hello

# 单参
x = lambda a: a + 10
print(x(5))  # 输出: 15

# 多参
x = lambda a, b: a * b
print(x(2, 3))  # 输出: 6
x = lambda a, b, c: a + b + c
print(x(1, 2, 3))  # 输出: 6
```

### 7.3 常用场景：配合 `map()`、`filter()`、`sorted()`

均为 Python 内置高阶函数，常搭配 `lambda` 使用，多用于**序列处理**。

| 函数         | 作用                  | 返回值                  |
| ---------- | ------------------- | -------------------- |
| `map()`    | 对序列每个元素做**统一映射处理**  | map 对象（需转列表 / 元组）    |
| `filter()` | 按条件**筛选元素**，保留符合条件项 | filter 对象（需转列表 / 元组） |
| `sorted()` | 对序列**排序**，生成新序列     | 排序后的新列表              |

#### 7.3.1 map () 映射

**语法**：`map(函数, 可迭代对象)`

- 依次把每个元素传入函数处理，返回新结果

```python
lst = [1,2,3]
res = list(map(lambda x: x*2, lst))
print(res)  # [2, 4, 6]
```

#### 7.3.2 filter () 过滤

**语法**：`filter(判断函数, 可迭代对象)`

- 只保留函数返回 `True` 的元素

```python
lst = [1,2,3,4]
res = list(filter(lambda x: x%2==0, lst))
print(res)  # [2, 4]
```

#### 7.3.3 sorted () 排序

**语法**：`sorted(序列, key=排序规则, reverse=False)`

- 不修改原数据，**返回新列表**
- `reverse=True` 降序，默认升序

```python
lst = [3,1,4,2]
# 基础排序
print(sorted(lst))  # [1, 2, 3, 4]

# 自定义规则：按绝对值排序
nums = [-3, 1, -2]
print(sorted(nums, key=lambda x: abs(x)))  # [1, -2, -3]
```

## 8. 嵌套函数与闭包

### 8.1 嵌套函数

定义：函数内部定义函数。

- 内层函数只能在外层函数**内部调用**
- 可以访问外层函数的变量

```python
def outer():
    print("外层")
    def inner():  # 嵌套函数
        print("内层")
    inner()       # 内部调用

outer()
# 输出：
# 外层函数
# 内层函数
```

### 8.2 闭包

内层用外层变量 + 外层返回内层，**带走并保存变量**

（1）闭包三要素

- 有**嵌套函数**
- 内层函数**使用了外层函数的变量**
- 外层函数**返回内层函数对象**

（2）闭包作用

**保存外层函数的变量状态**，函数退出后变量依然存在。

（3）极简示例

```python
def outer(x):
    def inner():
        return x + 10  # 内层使用外层变量
    return inner       # 外层返回内层函数

f = outer(5)
print(f())  # 15
```

## 9. 装饰器（面试常问）

### 9.1 定义

**作用**：不改动原函数代码，动态扩展功能。

**本质**：装饰器是一个函数，接收一个函数作为参数，返回一个新函数。

**语法**：`@装饰器名` 等价于 `原函数 = 装饰器(原函数)`

### 9.2 示例：

（1）这一段是 装饰器

这部分和你的业务代码没关系，是通用工具。

```python
def log(func):
    def wrapper(*args, **kwargs):
        # 日志开始 → 装饰器额外功能
        # 日志结束 → 装饰器额外功能
        # res = func(...) → 调用原函数
        # return res → 返回原函数结果
    return wrapper
```

（2）这一段是 原函数

```python
def add(a, b):
    return a + b
```

（3）这一段是 原函数的替身

```python
res = func(*args, **kwargs)  # 运行原函数
return res                   # 返回原函数的结果
```

（4）完整代码：

```python
# 装饰器（扩展功能）
def log(func):
    def wrapper(*args, **kwargs):
        # ===== 装饰器添加的功能 =====
        print("【装饰器】开始")

        # ===== 执行原函数 =====
        result = func(*args, **kwargs)

        # ===== 装饰器添加的功能 =====
        print("【装饰器】结束")
        return result
    return wrapper

# 原函数
@log
def say_hello(name):
    print(f"【原函数】你好, {name}")

# 调用
say_hello("张三") 
```

输出：

```shell
【装饰器】开始
【原函数】你好, 张三
【装饰器】结束
```

### 9.3 常用内置装饰器

| 装饰器             | 作用     | 特点                         |
| --------------- | ------ | -------------------------- |
| `@staticmethod` | 定义静态方法 | 无需`self/cls`，可通过类 / 对象直接调用 |
| `@classmethod`  | 定义类方法  | 首个参数为`cls`，可操作类属性          |
| `@property`     | 方法转为属性 | 调用时不加括号，常用于属性读取            |

**self = 对象自己**，**cls = 类自己**

Python 规定：**必须写第一个参数，而且习惯叫 self /cls**

#### 9.3.1 普通方法（必须带 self）

```python
class Person:
    def __init__(self, name):
        self.name = name  # 必须带 self

    def say_hello(self):  # 必须带 self
        print(f"我是 {self.name}")
```

#### 9.3.2 `@classmethod`类方法（必须带 cls）

```python
class Person:
    @classmethod
    def factory(cls):  # 必须带 cls
        return cls("张三")
```

#### 9.3.3 `@staticmethod` 静态方法（都不用带）

```python
class Person:
    @staticmethod
    def add(a, b):  # 什么都不用带
        return a + b
```

#### 9.3.4 `@property` 属性装饰器（把方法变属性）

```python
class Demo:
    def __init__(self):
        self.__age = 20

    @property
    def age(self):  # 调用不用加 ()
        return self.__age

d = Demo()
print(d.age)  # 输出 20 （不是 d.age()）
```

### 9.4 常用应用场景

| 应用场景      | 作用说明                     |
| --------- | ------------------------ |
| 日志记录      | 统一记录函数名称、调用时机、入参等信息      |
| 耗时统计      | 计算函数执行时长，用于性能分析与优化       |
| 登录 / 权限校验 | 执行函数前校验登录状态、操作权限，拦截非法访问  |
| 数据缓存      | 缓存函数执行结果，避免重复计算，提升运行效率   |
| 异常捕获      | 统一捕获函数运行异常，集中处理报错，防止程序崩溃 |
| 接口限流      | 限制函数调用频率，防护接口被频繁请求       |

#### 9.4.1 日志记录（最常用）

记录函数调用、参数、执行状态。

```python
def log(func):
    def wrapper(*args):
        print(f"调用函数：{func.__name__}")
        return func(*args)
    return wrapper

@log
def add(a,b):
    return a+b

add(2,3)
# 输出：调用函数：add
#       5
```

#### 9.4.2 运行耗时统计

计算函数执行时长，用于性能排查。

```python
import time
def timer(func):
    def wrapper(*args):
        start = time.time()
        res = func(*args)
        print(f"耗时：{time.time()-start:.2f}s")
        return res
    return wrapper

@timer
def test():
    for i in range(1000000):
        pass

test()
```

#### 9.4.3 权限 / 登录校验

执行函数前判断是否登录、有无权限。

```python
is_login = False

def check_login(func):
    def wrapper():
        if not is_login:
            print("请先登录！")
            return
        func()
    return wrapper

@check_login
def edit():
    print("编辑内容")

edit()  # 输出：请先登录！
```

#### 9.4.4 结果缓存

重复调用时直接取缓存，避免重复计算，提升效率。

```python
cache = {}
def cache_func(func):
    def wrapper(n):
        if n in cache:
            return cache[n]
        res = func(n)
        cache[n] = res
        return res
    return wrapper

@cache_func
def calc(n):
    return n**2

print(calc(5))
print(calc(5)) # 第二次直接读缓存
```

#### 9.4.5 异常捕获

统一捕获函数报错，避免程序崩溃。

```python
def catch_err(func):
    def wrapper():
        try:
            func()
        except Exception as e:
            print(f"出错：{e}")
    return wrapper

@catch_err
def div():
    1/0

div() # 输出：出错：division by zero
```

## 10. 函数注解（Type Hints）

**作用**：给参数和返回值标注类型，提高代码可读性，**不强制校验类型**，传错类型程序也不会报错。

**语法：**

```python
def 函数名(参数1: 类型, 参数2: 类型) -> 返回值类型:
    函数体
```

示例：

```python
# 标注参数为字符串、整数，返回值为字符串
def greet(name: str, age: int) -> str:
    return f"{name}, {age}岁"

# 注解仅提示，类型不符仍可正常运行
greet("张三", "二十") 
```

## 11. 函数作为参数

函数可以**当成变量**，传给另一个函数。

```python
# 高阶函数：接收一个函数 + 一个值
def apply(func, value):
    return func(value)  # 调用传进来的函数

# 两个普通函数
def square(x):
    return x * x

def double(x):
    return x * 2

# 把函数当参数传进去
print(apply(square, 5))   # 25
print(apply(double, 5))   # 10
```

最常用场景：`map` / `filter` / `sorted`

这里 **lambda 就是函数参数**

```python
list(map(lambda x:x*2, [1,2,3]))
```
