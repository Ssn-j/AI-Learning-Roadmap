（4）访问：使用方括号 [] 来截取字符串。

```python
str1 = 'rabbit'

print(str1[2])  # b

print(str1[-5])  # a

# 切片：从索引 0 开始，到索引 3 结束（不包含3）
print(str1[0:3])  # rab

# 切片：从头开始，取到索引 3（不包含3）
print(str1[:3])  # rab

# 切片：从索引 3 开始，取到字符串末尾
print(str1[3:])  # bit

# 切片：省略起始和结束索引，相当于复制整个字符串
print(str1[:])   # rabbit

# 直接输出完整字符串
print(str1)   # rabbit
```

- index() 方法：找出**某个值第一个匹配项**的索引位置

语法：

```python
list.index(x[, start[, end]])
```

- **参数**
  
  - x-- 查找的对象。
  
  - start-- 可选，查找的起始位置。
  
  - end-- 可选，查找的结束位置。

- **返回值**：返回查找对象的索引位置，如果没有找到对象则抛出异常。

```python
lst = [10, 20, 'red', 20, 40]

# 1. 基础用法：找元素第一次出现的下标

# 2. 从 start 位置开始找（包含start）
print(lst.index(20, 2))  # 3 → 从下标2开始找，找到下标3

print(lst.index('red', 0, 3))  # 2 → 在下标0~3之间找
```

-    参数

  - reverse -- 排序规则，`reverse = True`降序，`reverse = False`升序（默认）。

- 返回值
  
  - **原地修改原列表**，无返回值

```python
# 2. key 参数用法
# 按字符串长度排序
str_lst = ["apple", "hi", "banana"]
str_lst.sort(key=len)  # 以元素长度为依据排序
```

```python
print(max(list3))    # 报错
```

- min(list)：返回列表元素最小值，同max(list)

- list(iterable)：将序列转换为列表
  
  - **参数 iterable**：
    
    类型： 可迭代对象（元组、字符串、字典、集合等）
    
    描述： 要转换为列表的可迭代对象。
  
  - **返回值**： 返回一个列表。
  
  - **特殊值**： 不带参数返回空列表 `[]`。

```python
# 从元组转换为列表：保持元素顺序
t = (1, 2, 3, 4, 5)
lst = list(t)
print(lst)        # 输出: [1, 2, 3, 4, 5]

# 从字符串转换：每个字符成为列表的一个元素
s = "hello"
lst = list(s)
print(lst)        # 输出: ['h', 'e', 'l', 'l', 'o']

# 从字典转换（只取键）：字典转换只包含键
d = {"a": 1, "b": 2, "c": 3}
lst = list(d)
print(lst)        # 输出: ['a', 'b', 'c']

# 从集合转换
s = {3, 1, 2}
lst = list(s)
print(lst)        # 输出: [1, 2, 3]（自动排序）

# 空列表
lst = list()
print(lst)        # 输出: []
```
