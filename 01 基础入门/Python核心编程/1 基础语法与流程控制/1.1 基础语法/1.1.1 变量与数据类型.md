   1.Python中创建变量的方法是赋值

### 1.3 type()函数

## 2 数据类型
### 2.1 数字(Number)
Python 数字数据类型用于存储数值。

数据类型是不允许改变的，若改变数字数据类型的值，将重新分配内存空间。

#### 2.1.1 Number对象的创建与删除
（1）Number对象的创建：赋值

（2）Number对象的删除：del语句
del var1[,var2[,var3[....,varN]]]
可以通过使用del语句删除单个或多个对象的引用

#### 2.1.2 Python 支持三种不同的数值类型：

整型(int)：整型或整数，是正或负整数，不带小数点。Python3 整型是没有限制大小的，可以当作 Long 类型使用，所以 Python3 没有 Python2 的 Long 类型。布尔(bool)是整型的子类型。

浮点型(float)：浮点型由整数部分与小数部分组成，浮点型也可以使用科学计数法表示（2.5e2 = 2.5 x 102 = 250）

复数(complex)：复数由实数部分和虚数部分构成，可以用a + bj,或者complex(a,b)表示，复数的实部a和虚部b都是浮点型。

#### 2.1.3 Python 数字类型转换
有时候，我们需要对数据内置的类型进行转换，数据类型的转换，你只需要将数据类型作为函数名即可。

int(x) 将x转换为一个整数。

float(x) 将x转换到一个浮点数。

complex(x) 将x转换到一个复数，实数部分为 x，虚数部分为 0。

complex(x, y) 将 x 和 y 转换到一个复数，实数部分为 x，虚数部分为 y。x 和 y 是数字表达式。
