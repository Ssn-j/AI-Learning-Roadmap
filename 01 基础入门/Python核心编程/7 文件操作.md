# 文件操作

文件操作让程序能够读写硬盘上的数据，实现数据持久化。

## 1. 打开文件：`open()`

**注意：** 使用 open() 方法一定要保证关闭文件对象，即调用 close() 方法。

**基本语法**：

```python
file = open(文件路径, 模式)
```

**完整语法：**

```python
open(file, mode='r', buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)
```

**参数说明:**

- file: 必需，文件路径（相对或者绝对路径）。
- mode: 可选，文件打开模式
- buffering: 设置缓冲
- encoding: 一般使用utf8
- errors: 报错级别
- newline: 区分换行符
- closefd: 传入的file参数类型
- opener: 设置自定义开启器，开启器的返回值必须是一个打开的文件描述符。

**mode常用参数：**

| 模式    | 说明       | 文件不存在            | 文件存在            |
| ----- | -------- | ---------------- | --------------- |
| `'r'` | 只读（默认）   | 报错               | 正常读取            |
| `'w'` | 写入       | 创建               | 清空后写入           |
| `'a'` | 追加       | 创建               | 末尾追加            |
| `'x'` | 创建       | 创建               | 报错              |
| `'b'` | 二进制模式    | 与 `r`/`w`/`a` 组合 | 如 `'rb'`、`'wb'` |
| `'t'` | 文本模式（默认） | 与 `r`/`w`/`a` 组合 | 如 `'rt'`、`'wt'` |

**指定编码：**

```python
# 文本文件必须指定编码，常用 utf-8
file = open("data.txt", "r", encoding="utf-8")
```

## 2. 读取文件

| 方法            | 说明      | 返回值       |
| ------------- | ------- | --------- |
| `read()`      | 读取整个文件  | 字符串       |
| `read(size)`  | 读取指定字节数 | 字符串       |
| `readline()`  | 读取一行    | 字符串（含换行符） |
| `readlines()` | 读取所有行   | 列表        |

示例：

```python
# 读取整个文件

with open("data.txt", "r", encoding="utf-8") as f:
 content = f.read()
 print(content)

# 逐行读取（推荐，适合大文件）

with open("data.txt", "r", encoding="utf-8") as f:
 for line in f: # 直接遍历文件对象
 print(line.strip()) # strip() 去掉换行符

# 读取所有行到列表

with open("data.txt", "r", encoding="utf-8") as f:
 lines = f.readlines()
 for line in lines:
 print(line)
```

## 3. 写入文件

| 方法                 | 说明               |
| ------------------ | ---------------- |
| `write(str)`       | 写入字符串            |
| `writelines(list)` | 写入字符串列表（不会自动加换行） |

示例：

```python
# 写入字符串

with open("output.txt", "w", encoding="utf-8") as f:
 f.write("Hello, World!\n")
 f.write("第二行内容")

# 写入列表

lines = ["第一行\n", "第二行\n", "第三行\n"]
with open("output.txt", "w", encoding="utf-8") as f:
 f.writelines(lines)

# 追加内容

with open("log.txt", "a", encoding="utf-8") as f:
 f.write("新的一行日志\n")
```

## 4. `with` 语句（推荐）

**为什么用 `with`**：自动关闭文件，即使发生异常也会正确关闭。

```python
# 不推荐：需要手动关闭
file = open("data.txt", "r")
content = file.read()
file.close()  # 容易忘记写

# 推荐：自动关闭
with open("data.txt", "r", encoding="utf-8") as file:
    content = file.read()
# 离开 with 代码块后，文件自动关闭
```

**同时操作多个文件**：

```python
复制文件

with open("source.txt", "r") as source, open("dest.txt", "w") as dest:
 content = source.read()
 dest.write(content)
```

## 5. 文件指针操作

| 方法                     | 说明              |
| ---------------------- | --------------- |
| `tell()`               | 返回当前文件指针位置（字节数） |
| `seek(offset, whence)` | 移动文件指针          |

示例：

```python
with open("data.txt", "r") as f:
    print(f.tell())  # 0，开头位置
    f.read(5)
    print(f.tell())  # 5，移动了5个字节

    f.seek(0)  # 回到文件开头
    content = f.read()
```

## 6. 二进制文件操作

```python
# 读取图片
with open("image.jpg", "rb") as f:
    data = f.read()

# 写入图片副本
with open("image_copy.jpg", "wb") as f:
    f.write(data)
```

## 7. 路径操作

| 对比维度  | os /os.path（传统） | pathlib（现代推荐） |
| ----- | --------------- | ------------- |
| 编程风格  | 函数式写法           | 面向对象写法        |
| 跨平台   | 支持，但需手动处理路径     | 自动适配系统，路径更安全  |
| 代码可读性 | 一般，函数调用繁琐       | 优秀，直观简洁       |
| 适用场景  | 老旧项目、兼容旧代码      | 新项目、日常开发首选    |
| 简洁性   | 代码较长            | 代码更短更清晰       |

- `os.path`

导入os模块：

```pyrhon
import os
```

方法：

| 方法                      | 功能            | 示例代码                                |
| ----------------------- | ------------- | ----------------------------------- |
| `os.getcwd()`           | 获取当前工作目录      | `os.getcwd()`                       |
| `os.chdir(path)`        | 切换工作目录        | `os.chdir("D:/test")`               |
| `os.listdir(path)`      | 列出目录内容        | `os.listdir(".")`                   |
| `os.mkdir(path)`        | 创建空目录         | `os.mkdir("new_folder")`            |
| `os.rmdir(path)`        | 删除空目录         | `os.rmdir("new_folder")`            |
| `os.remove(path)`       | 删除文件          | `os.remove("data.txt")`             |
| `os.rename(src,dst)`    | 重命名文件 / 目录    | `os.rename("old.txt","new.txt")`    |
| `os.path.exists(path)`  | 判断文件 / 目录是否存在 | `os.path.exists("data.txt")`        |
| `os.path.isfile(path)`  | 判断是否为文件       | `os.path.isfile("data.txt")`        |
| `os.path.isdir(path)`   | 判断是否为目录       | `os.path.isdir("my_folder")`        |
| `os.path.getsize(path)` | 获取文件大小 (字节)   | `os.path.getsize("data.txt")`       |
| `os.path.join()`        | 跨平台拼接路径       | `os.path.join("folder","file.txt")` |
| `os.getenv(key)`        | 获取环境变量        | `os.getenv("HOME")`                 |
| `os.system(cmd)`        | 执行系统命令        | `os.system("dir")`                  |

- **路径操作（`pathlib`，更现代）**：

`pathlib` 是 Python3.4+ 推荐的现代路径工具，语法更简洁、面向对象，日常开发高频使用

方法：

| 用法                        | 功能           | 示例                                           |
| ------------------------- | ------------ | -------------------------------------------- |
| `Path(路径)`                | 创建路径对象       | `path = Path("data/my_file.txt")`            |
| `read_text(encoding)`     | 读取文本内容       | `path.read_text(encoding="utf-8")`           |
| `write_text(内容,encoding)` | 写入文本内容       | `path.write_text("Hello", encoding="utf-8")` |
| `name`                    | 获取完整文件名（含后缀） | `path.name`                                  |
| `stem`                    | 获取文件名（不含后缀）  | `path.stem`                                  |
| `suffix`                  | 获取文件后缀       | `path.suffix`                                |
| `parent`                  | 获取上级目录       | `path.parent`                                |
| `iterdir()`               | 遍历当前目录（不递归）  | `for f in Path(".").iterdir(): print(f)`     |
| `rglob(匹配规则)`             | 递归查找指定文件     | `for f in Path(".").rglob("*.py"): print(f)` |

示例：

```python
from pathlib import Path

# 创建路径对象
path = Path("data/my_file.txt")
# 读写文件
content = path.read_text(encoding="utf-8")
path.write_text("Hello", encoding="utf-8")
# 获取文件属性
print(path.name, path.stem, path.suffix, path.parent)
# 遍历目录
for file in Path(".").iterdir():
    print(file)
# 递归查找py文件
for file in Path(".").rglob("*.py"):
    print(file)
```

## 8. CSV 文件读写

| 操作  | 核心代码                              |
| --- | --------------------------------- |
| 读取  | `csv.reader(f)` 逐行读取数据            |
| 写入  | `csv.writer(f)`，`writerow()` 写入单行 |

示例：

```python
import csv

# 读取 CSV

with open("data.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)

# 写入 CSV

with open("output.csv", "w", encoding="utf-8", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(["姓名", "年龄", "城市"])
    writer.writerow(["张三", 20, "北京"])
```

## 9. JSON 文件读写

| 操作  | 核心代码                        |
| --- | --------------------------- |
| 写入  | `json.dump(数据, 文件句柄)`       |
| 读取  | `json.load(文件句柄)` 转为字典 / 列表 |

示例：

```python
import json

# 写入 JSON

data = {"name": "张三", "age": 20, "city": "北京"}
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 读取 JSON

with open("data.json", "r", encoding="utf-8") as f:
    data = json.load(f)
    print(data["name"])  # 张三
```
