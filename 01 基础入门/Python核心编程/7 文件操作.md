# 文件操作

文件操作让程序能够读写硬盘上的数据，实现数据持久化。

## 1. 打开文件：`open()`

**基本语法**：

```python
file = open(文件路径, 模式, encoding="编码")
```

**完整语法：**

```python
open(file, mode='r', buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)
```

**参数说明:**

- file: 必需，文件路径（相对或者绝对路径）。
- mode: 可选，文件打开模式
- buffering: 设置缓冲
- encoding: 一般使用utf8
- errors: 报错级别
- newline: 区分换行符
- closefd: 传入的file参数类型
- opener: 设置自定义开启器，开启器的返回值必须是一个打开的文件描述符。

**mode常用参数：**

| 模式    | 说明       | 文件不存在            | 文件存在            |
| ----- | -------- | ---------------- | --------------- |
| `'r'` | 只读（默认）   | 报错               | 正常读取            |
| `'w'` | 写入       | 创建               | 清空后写入           |
| `'a'` | 追加       | 创建               | 末尾追加            |
| `'x'` | 创建       | 创建               | 报错              |
| `'b'` | 二进制模式    | 与 `r`/`w`/`a` 组合 | 如 `'rb'`、`'wb'` |
| `'t'` | 文本模式（默认） | 与 `r`/`w`/`a` 组合 | 如 `'rt'`、`'wt'` |

**指定编码：**

```python
# 文本文件必须指定编码，常用 utf-8
file = open("data.txt", "r", encoding="utf-8")
```

## 2. 读取文件

| 方法            | 说明      | 返回值       |
| ------------- | ------- | --------- |
| `read()`      | 读取整个文件  | 字符串       |
| `read(size)`  | 读取指定字节数 | 字符串       |
| `readline()`  | 读取一行    | 字符串（含换行符） |
| `readlines()` | 读取所有行   | 列表        |

示例：

```python
# 读取整个文件

with open("data.txt", "r", encoding="utf-8") as f:
 content = f.read()
 print(content)

# 逐行读取（推荐，适合大文件）

with open("data.txt", "r", encoding="utf-8") as f:
 for line in f: # 直接遍历文件对象
 print(line.strip()) # strip() 去掉换行符

# 读取所有行到列表

with open("data.txt", "r", encoding="utf-8") as f:
 lines = f.readlines()
 for line in lines:
 print(line)
```

## 3. 写入文件

| 方法                 | 说明               |
| ------------------ | ---------------- |
| `write(str)`       | 写入字符串            |
| `writelines(list)` | 写入字符串列表（不会自动加换行） |

示例：

```python
# 写入字符串

with open("output.txt", "w", encoding="utf-8") as f:
 f.write("Hello, World!\n")
 f.write("第二行内容")

# 写入列表

lines = ["第一行\n", "第二行\n", "第三行\n"]
with open("output.txt", "w", encoding="utf-8") as f:
 f.writelines(lines)

# 追加内容

with open("log.txt", "a", encoding="utf-8") as f:
 f.write("新的一行日志\n")
```

## 4. `with` 语句（推荐）

**为什么用 `with`**：自动关闭文件，即使发生异常也会正确关闭。

```python
# 不推荐：需要手动关闭
file = open("data.txt", "r")
content = file.read()
file.close()  # 容易忘记写

# 推荐：自动关闭
with open("data.txt", "r", encoding="utf-8") as file:
    content = file.read()
# 离开 with 代码块后，文件自动关闭
```

**同时操作多个文件**：

```python
复制文件

with open("source.txt", "r") as source, open("dest.txt", "w") as dest:
 content = source.read()
 dest.write(content)
```

## 5. 文件指针操作

| 方法                     | 说明              |
| ---------------------- | --------------- |
| `tell()`               | 返回当前文件指针位置（字节数） |
| `seek(offset, whence)` | 移动文件指针          |

示例：

```python
with open("data.txt", "r") as f:
    print(f.tell())  # 0，开头位置
    f.read(5)
    print(f.tell())  # 5，移动了5个字节
    
    f.seek(0)  # 回到文件开头
    content = f.read()
```

## 6. 二进制文件操作

```python
# 读取图片
with open("image.jpg", "rb") as f:
    data = f.read()

# 写入图片副本
with open("image_copy.jpg", "wb") as f:
    f.write(data)
```

## 7. 路径操作（`os.path`）

```python
import os

# 检查文件/目录是否存在
os.path.exists("data.txt")

# 检查是否为文件
os.path.isfile("data.txt")

# 检查是否为目录
os.path.isdir("my_folder")

# 获取文件大小（字节）
os.path.getsize("data.txt")

# 获取目录下的文件列表
os.listdir(".")

# 创建目录
os.mkdir("new_folder")

# 删除文件
os.remove("data.txt")

# 重命名
os.rename("old.txt", "new.txt")

# 拼接路径（跨平台）
path = os.path.join("folder", "subfolder", "file.txt")


```

**路径操作（`pathlib`，更现代）**：

```python
from pathlib import Path
path = Path("data/my_file.txt")

# 读取/写入

path.read_text(encoding="utf-8")
path.write_text("Hello", encoding="utf-8")

# 属性

path.name # "my_file.txt"
path.stem # "my_file"
path.suffix # ".txt"
path.parent # "data"

# 遍历目录

for file in Path(".").iterdir():
 print(file)

# 递归查找所有 .py 文件

for file in Path(".").rglob("*.py"):
 print(file)

```

## 8. CSV 文件读写

```python
import csv

# 读取 CSV

with open("data.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)

# 写入 CSV

with open("output.csv", "w", encoding="utf-8", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(["姓名", "年龄", "城市"])
    writer.writerow(["张三", 20, "北京"])

```

## 9. JSON 文件读写



```python
import json

# 写入 JSON

data = {"name": "张三", "age": 20, "city": "北京"}
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 读取 JSON

with open("data.json", "r", encoding="utf-8") as f:
    data = json.load(f)
    print(data["name"])  # 张三

```
