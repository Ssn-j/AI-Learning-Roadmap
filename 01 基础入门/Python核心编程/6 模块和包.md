# 模块与包

模块是 Python 组织代码的基本单位。随着项目变大，把代码拆分成多个文件（模块）是必须的。

## 1. 模块（Module）

**定义**：一个 `.py` 文件就是一个模块，模块名就是文件名（不含 `.py`）。

### 1.1 导入模块

```python

# 方式1：导入整个模块

import math
print(math.sqrt(16))  # 输出：4.0

# 方式2：导入特定函数

from math import sqrt, pi
print(sqrt(16))  # 输出：4.0，不需要 math. 前缀
print(pi)        # 输出：3.14159...

# 方式3：导入所有（不推荐）

from math import *
print(sin(0))    # 输出：0.0，但不知道 sin 从哪来的

# 方式4：起别名

import numpy as np
from datetime import datetime as dt

### 1.2 导入自己写的模块

假设有两个文件在同一目录：

**`utils.py`**：

python

def greet(name):
    return f"你好, {name}"
PI = 3.14159

**`main.py`**：

python

import utils
print(utils.greet("张三"))  # 输出：你好, 张三
print(utils.PI)             # 输出：3.14159

### 1.3 模块搜索顺序

Python 导入模块时，按以下顺序查找：

1. 当前目录

2. 环境变量 `PYTHONPATH` 指定的目录

3. Python 安装目录（标准库）

4. 第三方库安装目录（site-packages）

python

import sys
print(sys.path)  # 查看所有搜索路径

### 1.4 `if __name__ == "__main__"`

**作用**：判断当前文件是被直接运行，还是被作为模块导入。

python

# my_module.py

def add(a, b):
    return a + b
if __name__ == "__main__":
    # 只有直接运行这个文件时才会执行
    print("测试:", add(3, 5))

| 运行方式                  | `__name__` 的值 |
| --------------------- | ------------- |
| `python my_module.py` | `"__main__"`  |
| `import my_module`    | `"my_module"` |

## 2. 包（Package）

**定义**：包含 `__init__.py` 文件的目录，用于组织多个模块。

### 目录结构

text

my_package/
├── __init__.py      # 可以为空，表示这是一个包
├── module_a.py
└── module_b.py

### 导入包中的模块

python

# 方式1

import my_package.module_a
my_package.module_a.some_function()

# 方式2

from my_package import module_a
module_a.some_function()

# 方式3

from my_package.module_a import some_function
some_function()

### `__init__.py` 的作用

- 标识一个目录是 Python 包

- 可以在其中定义包的初始化代码

- 可以控制 `from package import *` 导入的内容

python

# __init__.py

__all__ = ["module_a", "module_b"]  # 限制 from my_package import * 的导入范围

## 3. 常用模块导入模式

| 场景         | 推荐写法                        | 原因       |
| ---------- | --------------------------- | -------- |
| 使用模块中的少量函数 | `from math import sqrt, pi` | 代码简洁     |
| 使用模块中的大量函数 | `import math`               | 避免命名冲突   |
| 模块名较长      | `import numpy as np`        | 方便书写     |
| 自定义工具模块    | `import utils`              | 保持命名空间清晰 |

## 4. 避免循环导入

**问题**：两个模块互相导入对方。

**a.py**：

python

import b
def func_a():
    return b.func_b()

**b.py**：

python

import a
def func_b():
    return a.func_a()

**解决方案**：

- 将公共代码提取到第三个模块

- 在函数内部延迟导入

python

# 延迟导入

def func_a():
    from b import func_b
    return func_b()
